Apa itu CL-TB?

CB-TB merupakan sistem berbasis website yang memanfaatkan konsep machine learning untuk mengklasifikasikan hasil gambar chest x-ray ke dalam kategori normal atau tuberkulosis. Sistem ini dibuat untuk membantu dokter dan para ahli dalam mendiagnosis hasil chest x-ray pasien.

Petunjuk Penggunaan CL-TB

Pertama-tama, pilih menu Predict. Pada bagian menu tersebut, tekan tombol Pilih File untuk mengunggah gambar chest x-ray. Kemudian tekan tombol Predict image untuk melihat hasil prediksi gambar.
